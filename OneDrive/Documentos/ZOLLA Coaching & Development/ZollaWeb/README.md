# ZOLLA | Coaching & Development

Este es el nuevo sitio web de ZOLLA, construido con **Next.js 14**, **Tailwind CSS** y **Framer Motion** para ofrecer una experiencia premium, dinámica y profesional.

## Características
- **Diseño Premium**: Estética moderna con tipografía serif elegante (EB Garamond) y sans-serif limpia (Inter).
- **Animaciones**: Micro-interacciones y transiciones suaves usando Framer Motion.
- **Responsive**: Totalmente optimizado para móviles, tablets y escritorio.
- **Optimizado para Vercel**: Estructura lista para despliegue inmediato.
- **SEO**: Metadatos configurados para mejorar la visibilidad en buscadores.

## Desarrollo Local

1. Instalar dependencias:
   ```bash
   npm install
   ```

2. Iniciar el servidor de desarrollo:
   ```bash
   npm run dev
   ```

3. Abrir [http://localhost:3000](http://localhost:3000) en el navegador.

## Despliegue a GitHub

1. Inicializar repositorio git:
   ```bash
   git init
   ```
2. Agregar archivos:
   ```bash
   git add .
   ```
3. Primer commit:
   ```bash
   git commit -m "Initial commit: Rediseño Premium de ZOLLA"
   ```
4. Crear un repositorio en GitHub y seguir las instrucciones para subirlo:
   ```bash
   git remote add origin https://github.com/TU_USUARIO/NOMBRE_REPOSITORIO.git
   git branch -M main
   git push -u origin main
   ```

## Despliegue a Vercel

1. Ve a [vercel.com](https://vercel.com).
2. Haz clic en **Add New** > **Project**.
3. Importa tu repositorio de GitHub recién creado.
4. Vercel detectará automáticamente que es un proyecto de Next.js.
5. Haz clic en **Deploy**.
